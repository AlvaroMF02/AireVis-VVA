# 🌿 AireVis

AireVis es un proyecto de análisis y visualización de datos ambientales centrado en la calidad del aire y los niveles de polen. El objetivo es transformar datos abiertos en información clara y visual mediante procesos de limpieza, análisis y dashboards interactivos.

El proyecto utiliza datos históricos del servicio CAMS para mostrar tendencias relacionadas con contaminación ambiental y polen en Villanueva del Arzobispo (Jaén, España).

---

# Características

* Dashboard interactivo para visualización de datos ambientales.
* Datos históricos de contaminación y polen.
* Procesamiento y limpieza de datos con Python.
* Visualizaciones orientadas a análisis temporal.
